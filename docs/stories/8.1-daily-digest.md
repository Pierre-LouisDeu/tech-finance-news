# Story 8.1: Daily News Digest

## Epic
Epic 8: Daily Digest & Dokploy Deployment

## Description
Générer un résumé quotidien condensé de toutes les actualités traitées dans la journée et le pusher vers Notion comme une page spéciale "Digest".

## Acceptance Criteria
- [x] Récupérer tous les articles pushés aujourd'hui depuis la base SQLite
- [x] Agréger les résumés courts en un digest structuré par thème/source
- [x] Générer un résumé global via GPT (points clés de la journée)
- [x] Créer une page Notion "Daily Digest" avec date du jour
- [x] Le digest inclut: date, nombre d'articles, résumé global, liste des articles

## Technical Notes
- Nouveau module `src/digest/` pour la logique de digest
- Utiliser les summaries existants pour éviter des appels API supplémentaires
- Page Notion avec tag "Digest" pour différencier des articles individuels
- Query DB: articles avec `pushed` status et `processed_at` = today

## Dependencies
- Story 5.1, 5.2 (Notion Integration)
- Story 4.1, 4.2 (Summarizer)

Points: 3
